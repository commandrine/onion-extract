Simple Python script to index and download files (e.g. image) from an Onion URL.

Install the pre-requisites using the "requirements.txt" file.

Minor enhacements: 
<p>• 	File extension filtering</p>
<p>• 	MIME type validation</p>
<p>• 	SHA256 hash logging</p>
<p>• 	Timestamped audit log</p> 

Example: python3 onion-extract.py -o [local folder to download to] [onion URL]
